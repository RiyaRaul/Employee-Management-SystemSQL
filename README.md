# Employee Management System

## Project Description
This project is a simple **Employee Management System** created using **MariaDB**.
It stores and manages employee details, department information, projects, salary and client data.

The project is designed to understand:
1.Database tables
2.Primary Key and Foreign Key
3.Basic SQL queries



## Technologies Used
1.MariaDB
2. SQL
3.GitHub


## Database Tables
The database contains the following tables:
1. emp (Employee details)
2. emp_company (Employee job details)
3. department
4. project
5. emp_project
6.salary
7. address
8. qualification
9.client



## Concepts Used
1.Primary Key (PK)
2.Foreign Key (FK)
3.JOIN queries
4.Aggregate functions




## Sample Queries
This project includes basic SQL queries such as:
1.SELECT
2.WHERE
3.JOIN
4.COUNT

Example:

SELECT Name FROM emp WHERE Gender = 'Female';
